5 — Data Model & Module Map

Hub State Shape

phase
"idle | context_gathering | persona_generation | interviewing | complete"
agents{}
id → {type, model_tier, system_prompt, messages[], status, metadata}
events[]
{event_type, agent_id, summary, data, timestamp}
outputs.brief
Research brief JSON (from context agent)
outputs.personas
{design_rationale, personas[{name, role, system_prompt, …}]}
outputs.interview_strategy
{interview_strategy, interviewer_system_prompt}
outputs.transcripts[]
{persona_id, status, turns[{role, content}]}
system_log[]
API call records (model, tokens, errors)
orchestrator_log[]
Decision records (event, decision, reasoning)

File → Purpose

server.py
FastAPI entry, wires routes + pub/sub subscribers
config.py
Loads config.json, .env, prompt files
core/hub.py
State CRUD — the only shared dependency
core/agent_runtime.py
create · send · log · emit · export
core/model_selector.py
Tier → model ID resolution
core/orchestrator.py
Background worker thread, action executor
api/*_routes.py
Thin REST wrappers over core
api/events.py
SSE streaming endpoint
prompts/*.md
Base prompts loaded at runtime from disk
src/*.jsx
React 4-panel UI (Conversation, Persona, Interview, Log)
